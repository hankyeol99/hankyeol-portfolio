Overview

프로젝트 개요

Story of Travel은 국내 여행지 사진을 중심으로 사용자가 새로운 장소를 발견하고, 마음에 드는 여행 스토리를 저장하거나 작가를 팔로우할 수 있는 웹 서비스입니다. 테마별, 계절별, 지역별 탐색 구조를 통해 사용자가 원하는 분위기와 목적에 맞는 여행지를 빠르게 찾을 수 있도록 설계했습니다.

Service Direction

서비스 방향

01

문제 정의

여행 전 장소를 찾을 때 정보가 흩어져 있어 분위기와 취향에 맞는 여행지를 한눈에 비교하기 어렵습니다.

02

서비스 방향

사용자가 업로드한 여행 사진을 중심으로 테마, 계절, 지역, 인기 순위 기반 탐색 경험을 제공합니다.

03

핵심 구조

홈 → 테마별 여행지 → 시즌별 스토리 → 지역별 스토리 → 스토리 상세 / 북마크 / 팔로우

Core Features

핵심 기능

테마별 여행지

자연, 축제, 식도락 등 여러 테마를 선택해 취향에 맞는 여행 스토리를 탐색할 수 있도록 구성했습니다.

시즌별 스토리

봄, 여름, 가을, 겨울 계절별 여행 사진을 모아볼 수 있도록 설계했습니다.

지역별 스토리

사용자가 업로드할 때 등록한 지역 정보를 기반으로 지역별 여행 스토리를 탐색할 수 있도록 구성했습니다.

Best Story

조회수와 좋아요 수를 기준으로 인기 여행 스토리를 순위 형태로 보여줍니다.
