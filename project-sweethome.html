Overview

프로젝트 개요

스윗홈은 말기 환자들이 자신의 통증 상태와 진통제 복용 이력을 체계적으로 기록할 수 있도록 돕는 모바일 앱입니다. 사용자는 통증 강도, 위치, 지속 시간, 악화 요인, 진통제 복용 여부를 기록할 수 있으며, 누적된 기록은 리포트 형태로 정리되어 환자와 보호자, 의료진이 통증 흐름을 쉽게 확인할 수 있도록 설계했습니다.

Core Features

핵심 기능

복잡한 의료 정보를 단순한 입력 흐름으로 정리하고, 통증 기록부터 복약 관리, 리포트 확인까지 이어지는 사용 흐름을 설계했습니다.

통증 기록

통증 강도, 위치, 원인, 지속 시간 등을 간단히 입력하고 기록할 수 있도록 설계했습니다.

진통제 관리

정기 복용 진통제와 속효성 진통제를 구분해 복용 이력을 관리할 수 있도록 구성했습니다.

통증 리포트

날짜별 통증 기록과 진통제 복용 이력을 캘린더와 목록 형태로 확인할 수 있도록 설계했습니다.

통증 가이드

통증 관리와 진통제 복용에 필요한 정보를 사용자가 쉽게 찾아볼 수 있도록 정리했습니다.
